Creer un project web maven : 
Source : http://www.mkyong.com/maven/how-to-create-a-web-application-project-with-maven/

mvn archetype:generate -DgroupId={project-packaging} -DartifactId={project-name} -DarchetypeArtifactId=maven-archetype-webapp -DinteractiveMode=false

commande mvn pour eclipse: mvn eclipse:eclipse -Dwtpversion=2.0
